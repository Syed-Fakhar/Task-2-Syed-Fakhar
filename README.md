# Expense Tracker Application

A simple Python-based application designed to help users track and manage their daily expenses. This project demonstrates core Python fundamentals alongside a dynamic multi-themed user interface.

## 🚀 Features

* **Add Expenses:** Log new expenditures with ease.
* **Delete Expenses:** Remove incorrect or outdated expense entries.
* **View Expenses:** Display a clear overview of all recorded spending.
* **Multi-Themed App:** Switch between different visual themes for a personalized experience.

## 🧠 Key Learnings

Building this project helped improve my understanding of:
* Python basics and data structures
* Implementing multi-themed user interfaces
* Functions, loops, and conditional logic
* Data tracking and state management

## 🛠️ Installation & Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```
2. **Navigate to the project directory:**
   ```bash
   cd expense-tracker-python
   ```
3. **Run the application:**
   ```bash
   python main.py
   ```
